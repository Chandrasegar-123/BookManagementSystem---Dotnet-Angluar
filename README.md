# Book Management System

## Overview
This is a simple full-stack application built using Angular and ASP.NET Core.
It allows users to add, view, update, and delete books.

## How to Run

### Backend
cd BookManagementApi

dotnet run

Runs on:
http://localhost:5056

### Frontend
cd book-management-ui

npm install

ng serve

Runs on:
http://localhost:4200
